# Getting Started
{: .no_toc }

Install VISTA, configure your AI provider, and start security testing in minutes.
{: .fs-6 .fw-300 }

---

## Overview

Getting started with VISTA involves three simple steps:

1. **[Install]({% link getting-started/installation.md %})** — Download or build the VISTA JAR and load it into Burp Suite
2. **[Configure]({% link getting-started/quick-start.md %})** — Set up your AI provider (OpenAI, Azure, or OpenRouter)
3. **[Test]({% link getting-started/quick-start.md %}#step-2-start-testing)** — Start analyzing traffic or use the AI Advisor

{: .tip }
> Want to use VISTA for **free**? See the [Free AI Setup]({% link getting-started/free-ai-setup.md %}) guide — no credit card required!
